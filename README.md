"# Birthday Blowout

An interactive birthday celebration page built with HTML, CSS, and JavaScript.

## What it does
- renders a fun birthday cake and candles
- listens for microphone input so users can "blow out" the candles
- plays a happy birthday melody
- shows confetti and a custom popup message after the blow
- works responsively on smaller screens

## How to use
1. Open `index.html` in a modern browser.
2. Click **Start Microphone & Listen** and allow mic access.
3. Enter a name, make a wish, then blow into the microphone.
4. Press **Reset / Relight Candles** to try again.

## Improvements made
- improved audio blow detection using real microphone waveform data
- added keyboard-friendly focus styling
- enabled proper text selection for inputs and buttons
- added accessible status updates and cleanup logic
